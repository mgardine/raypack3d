Thank you for downloading the RAYPACK3D codebase. The code is provided as
is, with no warranty or guarantee.  Most of the functions require either
Antelope, raytrace3d, or both.

For a basic intro and list of available functions, please run the 
ray_viewcookbook.m function from a Matlab command line.

If you find a bug, please post it on the Google code website:
http://code.google.com/p/raypack3d
